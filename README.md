TensorFlow - Análisis de Imágenes y Machine Learning
Este repositorio contiene proyectos de machine learning y redes neuronales desarrollados con TensorFlow, enfocados en análisis de imágenes, clasificación y predicción de datos.

---

Proyectos incluidos

Clasificación de imágenes (Perros vs Gatos)
Modelo de red neuronal convolucional (CNN) entrenado para clasificar imágenes de perros y gatos.

Características:
- Uso de dataset Cats vs Dogs
- Preprocesamiento de imágenes (redimensionamiento y escala de grises)
- Modelo CNN con múltiples capas convolucionales
- Entrenamiento y validación del modelo
- Predicción sobre nuevas imágenes

---

Predicción (Pulgadas a Metros)
Modelo de regresión utilizando redes neuronales para convertir pulgadas a metros.

Características:
- Modelo simple con capa densa
- Entrenamiento con datos numéricos
- Visualización de la pérdida durante el entrenamiento
- Predicción de valores nuevos

---

Clasificación de basura con visión por camara
Sistema de análisis de imágenes mediante cámara para identificar y clasificar objetos de basura.

Objetivo:
- Detectar tipos de residuos mediante imágenes
- Clasificar objetos (plástico, papel,carton, organico,cristal)
- Aplicar modelos de computer vision en tiempo real

---

Tecnologías utilizadas
- Python
- TensorFlow / Keras
